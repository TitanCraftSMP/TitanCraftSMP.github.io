Häufig gestellte Fragen (FAQ)

Welche Version ist euer Server?

Die Hauptversion unseres Servers ist 1.20.1. Ihr könnt jedoch ab der Version 1.8 jede Version nutzen, um dem Server beizutreten.

Ist der Server für Bedrock?

Ja, unser Server unterstützt Bedrock-Edition. Ihr könnt sowohl mit Java Edition als auch mit Bedrock Edition dem Server beitreten.

Kann man sich bewerben?

Ja, Bewerbungen sind möglich. Aktuell suchen wir jedoch keine neuen Supporter.
